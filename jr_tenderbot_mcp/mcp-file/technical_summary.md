   - 拟采购劳务派遣和服务外包服务：

     - 劳务外包114人

   - 两年，合同一年一签
   - 第一年度合同到期前一个月双方无书面异议，可续签1次

## 来自章节: 第二章 投标人须知

技术要求内容摘录如下：

1. 投标报价（实质性要求）  
- 投标人的报价是其响应本项目要求的全部工作内容的价格体现，包括所有含税费用。  
- 每项报价内容只允许一个报价，选择性或可调整报价不被接受，视为无效投标。  
- 价格修正后需投标人加盖印章确认，否则投标无效。  
- 报价明显低于其他有效报价时，须提交合理性说明及可能的证明材料，不能证明合理性的投标视为无效。

2. 投标文件语言（实质性要求）  
- 投标文件及相关往来文件必须使用中文。  
- 外文资料必须逐一对应翻译成中文并加盖公章，可附于外文资料后面。  
- 关键内容未翻译影响评审者，相关资料视为无效材料，投标文件可能无效。  
- 翻译版本与原文不符时，以中文为准。  

3. 计量单位及投标货币（实质性要求）  
- 使用国家法定计量单位。  
- 以人民币报价。

4. 知识产权（实质性要求）  
- 投标产品及服务不得侵害第三方专利权、商标权等知识产权，因纠纷产生责任由投标人承担。  
- 招标人享有项目实施过程中产生的知识成果及知识产权。  
- 投标人自有知识产权需声明并提供证明，提供相关开发接口和技术文档，承诺免费技术支持及招标人免费使用权。  
- 使用非自有知识产权需合法授权并将费包含报价中，确保招标人永久使用权。  
- 提供的服务或产品不含未披露的需许可内容。

5. 投标文件格式  
- 严格按照招标文件第五章“投标文件格式”填写，非允许范围不得使用其他格式。  
- 无格式要求的自行编写。

6. 投标保证金（实质性要求）  
- 按规定必须交纳人民币10000元投标保证金。  
- 未及时足额缴纳视为无效投标。  
- 中标后5个工作日内无息退还（非中标无条件退还，特殊情况详招标文件）。  
- 特定情形下保证金不退还，包括撤回投标、中标后弃权、虚假资料等。

7. 投标有效期（实质性要求）  
- 有效期不少于90天，从投标截止日期起算。  
- 期满后可书面要求延长，拒绝不影响保证金。

8. 合同分包和转包（实质性要求）  
- 本项目不允许分包（具体标注为不允许）。  
- 严禁转包，转包行为视同拒绝履行合同，追究法律责任。

9. 履约保证金（实质性要求）  
- 中标后合同签订前交纳5万元履约保证金。  
- 不履行合同、提供假冒伪劣产品、降低配置或多次售后服务质量问题，可能导致履约保证金部分或全部不退还。

10. 合同履行（实质性要求）  
- 合同签订后，招标人有权依据招标文件对产品及解决方案做初期验证，验证不合格可终止合同并追究违约。  
- 合同双方务必严格履约，发生纠纷按《民法典》和合同约定处理。

11. 验收（实质性要求）  
- 按合同约定及相关行业标准、招标人制度严格验收。

12. 付款（实质性要求）  
- 按招标人相关规定及合同约定执行。

13. 针对提供货物的具体技术要求  
- 所有产品及辅材必须通过国家强制性产品认证并取得认证证书（涉及强制性认证范围产品）。  
- 产品及辅材符合国家强制性标准及行业标准。  
- 商品及快递包装必须符合法律法规，具备防潮、防震、防锈及适应长途运输能力，包装质量不足导致的货物损失由供应商承担。

14. 投标文件的印制和签署  
- 正本壹份，副本壹份，电子文档壹份。  
- 使用不褪色、耐久墨水书写或打印，逐页编码，A4规格纸张。

15. 投标文件密封要求  
- 文件正本、副本、开标一览表和电子文档分别密封，标注清晰。  
- 密封袋封口处粘贴牢固，盖投标人印章。  
- 未密封或密封不符要求文件不接受。

以上为文本中包含的所有与技术规格、功能要求、系统架构、开发规范、性能指标、测试要求相关的实质性技术内容和条款。

5.评标程序中对技术与资质的审查要求：

5.1 投标文件初审分资格审查和符合性审查：

- 资格审查中明确投标人应具备“履行合同所必需的设备和专业技术能力”，需书面声明材料证明；

- 特殊资格要求：供应商应具有有效的劳务派遣经营许可证和人力资源服务许可证。

5.1.2 符合性审查中需审查投标文件是否实质性响应招标文件第六章（采购需求）和第七章（合同条款）中的技术、服务要求，“必须全部响应，不得有负偏离”。

5.2 评标过程中的技术相关澄清和补正：

- 评标委员会可书面要求投标人对技术信息和数据资料等含义不明确、错误的内容做澄清、说明或补正；

- 澄清、说明或补正不得改变投标文件的实质性内容，不得影响公平竞争。

5.3 比较与评价中：

- 对技术、商务规定不符的非实质性偏离不作否定，但实质性偏离会导致投标无效。

5.4 复核中：

- 重点复核报价最低且符合技术要求的投标文件，及投标文件被认定无效的技术原因。

## 来自章节: 第五章 投标文件格式

技术要求相关内容摘录如下：

1. 投标人承诺函（第3.3节）
- 提供的所有服务或工程符合现行国家相关强制性标准、行业标准。
- 所有产品、辅材中属于《国家强制性产品认证目录》范围内的，均通过国家强制性产品认证并取得认证证书。
- 所有产品、辅材符合现行强制性国家和行业标准。
- 若国家或行业主管部门对采购产品的技术标准、质量标准和资格资质条件有强制性规定，必须符合。
- 商品包装和快递包装均符合相关法律法规要求，包装适应远距离运输、防潮、防震、防锈和防野蛮装卸，确保货物安全无损运抵指定地点。
- 由于包装不善导致的货物损失由投标人承担。

2. 项目要求应答表（第4.3节）
- 投标人应根据招标文件第六章中技术要求、服务要求、商务要求等内容及第七章合同条款进行逐条应答。
- 针对实质性技术要求，投标文件中存在负偏离的应答将被认定为无效投标。
- 未明确应答的条款视为接受并满足招标文件要求。
- 应答分为“满足”、“部分满足”、“不满足”，偏离说明填写“无偏离”、“正偏离”或“负偏离”，并附具体说明。
- 必须真实填写，不得虚假应答。
- 应答表多页须每页盖章。

3. 项目实施方案（服务类项目）（第4.5节）
- 投标人需提交项目实施方案，具体内容未详述，但属于技术相关资料。

4. 劳务派遣与劳务外包报价（第2.2节）
- 劳务派遣服务报价中，两年费用为固定价，管理费两年费用合计报价=两年费用+管理费。
- 劳务派遣服务管理费不超过30元/人/月。
- 外包服务费包含在外包人员费用中，大堂经理、网点营销助理、工勤人员及营销策划岗外包服务管理费不高于200元/月/人，其他管理费不高于120元/月/人。

5. 投标人的技术和专业能力声明（第3.2节声明事项第9条）
- 投标人声明具有履行合同所必需的设备和专业技术能力。

注：以上内容均摘自招标文件中的技术要求、服务要求、项目要求应答等章节，涉及产品标准、包装要求、技术响应须知、服务价格管理费限制及投标人技术能力认定等具体技术规范和要求。

## 来自章节: 第六章  采购需求

技术要求提取：

1. 劳务派遣人员费用构成及管理费：
   - 劳务派遣费用包含：劳动报酬、加班工资、差旅及里程补贴、劳动保护、业务计价及按国家规定由单位承担的社会保险费（基本养老、医疗、失业、工伤、生育保险等）。
   - 劳务派遣费用为固定报价（不可竞争费用），不含管理费。
   - 管理费报价不超过30元/人/月。
   - 实际费用按银行预算范围内实际结算。

2. 劳务外包服务费用构成与管理费限制：
   - 劳务外包费用包含：服务管理费、劳动报酬、劳动保护、业务计价及按国家相关规定承担的社会保险费。
   - 大堂经理、网点营销助理、工勤人员及营销策划岗外包管理费限200元/月/人，其他岗位管理费不高于120元/月/人。

3. 劳务人员提供及服务响应时间：
   - 根据用工方需要，原则上3个工作日内提供合适劳务人员。

4. 劳务人员劳动关系与管理：
   - 劳务人员必须身体健康，劳务公司负责办理合法劳动用工手续。
   - 劳务公司承担因招聘或管理不当产生的劳动纠纷或法律责任。
   - 建立和完善劳务人员管理制度，规范员工行为，并接受用工方监督。

5. 教育培训要求：
   - 入职培训覆盖：企业背景、职业道德、劳动用工特性、权利义务、社保知识、薪酬福利、安全消防、银行服务礼仪等。
   - 培训频次至少每季度一次。

6. 服务方案及应急预案：
   - 提供详尽的服务方案，包括但不限于：大堂经理外包、网点营销助理外包、炊事保洁外包、驾驶员派遣、工勤人员外包、应急预案。

7. 服务质量与管理规范：
   - 按相关规范制定服务和项目规范，保证服务质量及效率。
   - 应有项目实施人员配备、服务评价、风险控制、投诉处理、现场管理、业务考核、数据报表分析及管理流程。
   - 禁止因服务质量不达标导致媒体曝光、网络投诉。

8. 人员及设施配置要求：
   - 具备适应劳务代理业务的固定人员、配套设施及经营场所。

9. 部分岗位人员条件：
   - 大堂经理：
     - 招聘范围：泸州市四县三区常住人口，优先本地。
     - 学历：职高、中专及以上。
     - 年龄：18-35岁。
     - 身高要求：女≥1.55米，男≥1.65米。
     - 普通话标准，形象气质佳，五官端正，身体健康。
     - 服务、销售工作经验优先，熟练电脑及手机软件。
     - 机动车驾驶证C1以上者优先。
     - 遵纪守法，品行端正。
     - 提供身份证、户口本、毕业证、驾驶证复印件、《体检报告》、《无犯罪证明》、《无不良信用证明》。
