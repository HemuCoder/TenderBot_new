## 来自章节: 第一章 招标公告

评分要求相关内容（原文摘录）：

文本中未直接包含明确的评分标准、评分细则、加分项、减分项或废标条款的具体内容。以下为投标文件提交及资格要求等条款信息，可能涉及评分或评审基础：

- 资格要求：
  - 具有独立承担民事责任能力；
  - 具有良好的商业信誉和健全的财务会计制度；
  - 具有履行合同所必需的设备和专业技术能力；
  - 有依法缴纳税收和社会保障资金的良好记录；
  - 参加本次采购活动前三年内无重大违法记录；
  - 不得为失信被执行人、重大税收违法失信主体、政府采购严重违法失信行为记录名单内供应商；
  - 不得为四川农商联合银行采购“黑名单”企业或自然人及相关控制企业；
  - 投标人和其主要负责人前三年内无行贿犯罪记录；
  - 不接受联合体投标；
  - 本项目不得转包、不允许分包；
  - 供应商须持有效劳务派遣经营许可证和人力资源服务许可证。

- 投标文件提交要求：
  - 截止时间：2025年12月10日10:30（北京时间）；
  - 提交方式为线下递交纸质文件，地址明确；
  - 逾期或未送达指定地点的投标文件不予受理。

- 获取招标文件及澄清说明：
  - 招标文件获取期间及方式说明；
  - 若有澄清或修改，供应商应重新获取招标文件，否则后果自负。

总结：以上为涉及投标人资格条件和投标文件提交要求，属于投标评审的基本资格审查内容，但文本中未包含具体评分细则、评分标准或加减分以及废标规则。

若需评分标准、细则等内容，建议查阅招标文件第六章及相关评分办法或评审标准部分。

## 来自章节: 第二章 投标人须知

评分要求相关条款及描述摘录如下：

一、投标人须知附表中关于评分与评标规定  
- 序号4：评标方法  
  “综合评分法（详见第三章）”  

- 序号12：报价/分值精确度  
  “除招标文件明确规定外，所有数据项最多精确至小数点后2位，超出小数点位的数值采用四舍五入的方式进行保留。”

二、总则相关  
- 6.1条：招标文件的构成  
  包含“（三）评标办法”且指出评标办法是评标的重要依据。  
- 6.2条：招标文件响应要求  
  “投标人没有对招标文件全面做出实质性响应所产生的风险由投标人承担。没有按照招标文件要求作出实质性响应的投标文件将按无效投标处理。”

三、投标文件相关  
- 10条：投标报价（实质性要求）  
  “投标人对采购内容报价要求中的每一项报价内容只允许有一个报价，任何有选择或可调整的报价将不予接受，并按无效投标处理。”  
  “按照招标文件第三章评标办法规定进行价格修正的，修正后的报价经投标人加盖印章确认后产生约束力。未在规定时间内确认的，视同投标人不确认，其投标无效。”  
  “评标委员会认为投标报价明显低于其他合格投标人报价，有可能影响质量或履约的，应要求投标人进行书面说明并提供证明材料，不能证明合理性的按无效投标处理。”  

- 11条：投标文件语言（实质性要求）  
  “主要评分证明材料等未对应翻译成中文，影响评标的，视为无效材料，投标文件可能无效。”  

- 15条：投标保证金（实质性要求）  
  详述保证金数额、交纳、退还及不予退还的情形，不退还情形包括提交虚假资料、违法违规等，涉及评分有效资格。  

- 16条：投标有效期（实质性要求）  
  “投标有效期不得少于90天，短于此标准的投标被拒绝。未书面拒绝延长期限的，视为同意。”  

- 19条：投标文件修改和撤回（实质性要求）  
  “投标截止时间后不得修改或撤回投标文件。”  

四、开标及评标  
- 20.5条：开标时不符合要求的投标“开标一览表”相关信息异常，将记录并按招标文件规则进行评审。  

五、废标条款  
- 22条：废标情形  
  (1) 不足三家符合条件或不达到最低入围数量的合格、实质响应投标人；  
  (2) 出现影响采购公正的违法违规行为；  
  (3) 投标报价均超过采购预算且招标人不能支付；  
  (4) 采购任务因重大变故被取消。  
- 废标后须公告废标原因。  

六、定标与中标通知  
- 23条：定标遵循评标委员会推荐顺序确定中标人，招标人可基于正当理由调整或重新招标。  
- 25.4条：若中标投标文件应无效或违反法规，中标通知书将被宣布无效，重新确定中标人。  

七、纪律要求与违规处理影响评分资格  
- 33条：纪律要求  
  - 33.2.1条：投标人存在以下情形，投标文件无效或取消中标资格：  
    (1) 提供虚假材料谋取中标；  
    (2) 采取不正当手段排挤竞争对手；  
    (3) 串通投标、围标等行为；  
    (4) 行贿或提供不正当利益；  
    (5) 中标后拒绝履约等多项行为。  
  - 33.2.2条：违法失信行为列入黑名单，期限依据情节轻重不等，影响资格和评分权利。  

八、补充其他评分相关实质要求  
- 35.1条：需符合国家强制标准及行业标准。  
- 35.2条：涉及货物必须通过国家强制性产品认证，包装需满足法规及运输安全要求。  

综上，本项目评分要求包括：  
1. 综合评分法作评标方法，详见第三章。  
2. 投标报价严格限一报且报价精度限制，小数点后最多两位。  
3. 投标文件需实质响应招标文件，否则无效。  
4. 明显偏低报价需要投标人书面合理性说明，否定其报价无效。  
5. 语言文件及主要资料未中英文对应或翻译不全，影响评审有效性。  
6. 投标保证金缴纳及不退还情形作为投标资格及评分前提。  
7. 投标有效期不少于90天，否则无效。  
8. 招标文件及开标时资料异常或未如实提供导致评审无法正常进行，相关投标文件无效或废标。  
9. 违法违规或失信行为直接导致投标文件无效或取消中标，影响评分资格。  
10. 废标情形明确，涉及合格投标人不足、违法违规、超预算等。  
11. 要求符合技术和质量标准，涉及评审内容。  

以上条款即为本次招标项目评分相关的具体要求、加减分明示及废标情况。

## 来自章节: 第三章 评标办法

评分要求提取如下：

四、评标方法—综合评分法

| 评分项   | 具体内容             | 分值 | 备注                                                                                   |
|----------|----------------------|------|----------------------------------------------------------------------------------------|
| 投标价格 | 投标价格             | 30分 | 计分规则：以最低的合格报价为基准价，得分=基准价/投标报价×30，价格分满分30，最低0分。投标报价低于其他有效投标人报价算术平均价50%，且无法合理说明或提供证明的，可认定为低于成本报价竞标，做废标处理。 |
| 履约能力 | 类似项目业绩         | 25分 | 2022年1月1日以来，每有1个类似项目业绩得5分，最高25分。需提供合同关键页复印件、增值税发票复印件及查验截图，若无发票需书面说明并提供收款回单、业主证明或官网中标公告等佐证材料。所有证明加盖投标人公章，原件备查。业绩按格式列明。 |
| 履约能力 | 信誉                 | 18分 | 提供有效的证书得分：质量管理体系认证3分，环境管理体系认证3分，职业健康安全管理体系认证3分，信息安全管理体系认证3分，食品安全管理体系认证3分，社会责任管理体系认证3分；本项最多得18分。提供证书复印件加盖鲜章。 |
| 履约质量 | 实施团队             | 12分 | 1）项目负责人具有人力资源管理师证书得3分；2）其他管理人员中三级及以上企业人力资源管理师每人2分，最多4分；3）服务人员中一级劳动关系协调师每人3分，二级每人2分，最多5分。需提供身份证、资格证书复印件及查询截图，加盖鲜章；项目负责人提供缴纳养老保险证明或劳动合同复印件。 |
| 履约质量 | 服务方案             | 15分 | 人员管理制度（含五项制度）：完全符合给5分，每缺1项扣1分，每项内容错误或不满足扣0.5分；派遣外包方案（含六项方案）：完全符合给10分，每缺1项扣2.5分，每项错误扣1.25分。本项扣完为止。 |

五、评标程序—资格审查及符合性审查（合格/不合格的处理规则）

5.1 资格审查标准：

| 审查项                         | 通过条件及要求                                                                                     |
|-------------------------------|--------------------------------------------------------------------------------------------------|
| 投标人独立承担民事责任能力     | 营业执照（副本）、法人登记证（副本）或相关证明材料复印件，证件须在有效期内。                            |
| 书面声明材料                   | 包括：商业信誉和财务会计制度证明、履行合同设备及专业技术能力、缴纳税收和社保资金证明、无重大违法记录、未列入失信名单、无行贿犯罪记录、未被列入采购“黑名单”等声明材料，均需按招标文件要求提供。联合体所有成员均需符合。 |
| 其他资格要求                   | 不属于法律禁止参加投标的情形，非联合体或联合体提供协议书，按要求缴纳保证金，具备有效劳务派遣经营许可和人力资源服务许可证。 |

- 资格不符者投标文件无效；资格审查通过的供应商<3名或少于入围要求则采购失败。

5.1.2符合性审查标准：

| 审查项                         | 通过条件                                                                                         |
|-------------------------------|------------------------------------------------------------------------------------------------|
| 密封、签署、盖章               | 投标文件符合招标文件规定密封（加密）、签署、盖章要求。                                         |
| 符合采购文件实质性要求         | 语言、计量单位、报价货币、投标有效期、知识产权符合招标文件要求。                               |
| 投标报价                     | 报价应为全部工作内容的价格体现，不允许选择性调整报价，报价不得超过预算和最高限价，报价不唯一视为无效。报价明显低于其它合格投标可要求书面说明，无法合理说明作无效。 |
| 响应第六、第七章实质要求       | 投标文件实质性响应采购需求及合同条款中的要求，无负偏离。                                     |
| 其他证明材料                   | 法定代表人授权书及身份证明、法定代表人或主要负责人身份证复印件或护照复印件等按要求提供。         |

- 不符合任一项，投标文件视为无效。
- 符合性审查中对轻微缺陷（未盖章密封、未按要求分装、少量缺签）可视为不影响有效性，通过符合性审查。

五、评标过程中的解释、澄清及说明：

- 评标委员会可要求招标人或代理机构书面解释招标文件不明确事项，不得改变原义或影响公平。
- 针对投标文件含义不明、表述不一致、文字和计算错误，要求投标人书面澄清说明或补正，澄清内容不得改变投标文件实质性。
- 投标报价前后不一致按开标一览表为准，对大写小写金额、单价及总价不一致按规定修正，修正后须投标人确认，未确认视为无效。
- 多语言投标文件中文文本为准。
- 投标人应及时响应澄清要求，否者自行承担后果。
- 不得滥用澄清权，合理澄清仍不得直接作无效处理。

五、比较与评价：

- 对资格审查、符合性审查合格投标文件根据评标方法进行商务和技术评估。
- 以下非实质性偏离不得否决投标：技术说明含义不明确或内容不完整但及时澄清无效；未按要求装订或目录页码；非实质性偏离技术、商务条款；评标委员会认定的其他非实质性偏离。

五、复核：

- 评标委员会对评分汇总、拟推荐中标候选人、最低报价、中标文件无效认定进行重点复核。
- 代理机构现场监督并复核评审结果，发现错误或异常应书面建议评标委员会修改或重新评审。
- 不合规由招标人依法处理。

五、中标候选人推荐规则：

- 中标候选人数量根据采购需要定，按总得分由高到低排序3名。
- 采用最低评标价法，按报价由低到高排序，报价相同随机抽取确定。
- 采用综合评分法，按得分由高到低排序，得分相同按价格排序，价格相同再按“履约能力”、“履约质量”得分排序，仍相同随机抽取。
- 同核心产品品牌投标人，评标时只计得分最高的一家，并按上述规则确定。

五、评标报告编写：

- 包括招标公告信息、投标人名单、评标方法及标准、开标记录及无效投标名单及原因、评标结果和排序、中标建议、澄清说明情况、评标委员更换情况、报价合理性说明等。
- 评标委员签字确认，对异议应书面说明理由。

五、评标争议处理：

- 评标委员会争议以少数服从多数原则解决，但不得违法法规。
- 持不同意见成员应签署异议及理由，必要时向招标人或代理机构书面反映。

五、废标条款：

- 资格审查未通过投标人，投标文件无效。
- 符合性审查未通过，投标文件无效。
- 报价明显低于其他投标人50%，且不能合理说明或无相关证明材料的，响应废标。
- 投标报价错误且未能按规定确认的，投标无效。
- 提供虚假证明材料等违法违规行为将依法处理。
- 存在非法干预评标的违法行为相关责任将依法追究。
- 被列入采购“黑名单”或不符合资格认定的投标人参与投标，投标文件无效。

以上即为全文中涉及的评分标准、评分细则、加减分规则和废标条款等评分要求相关内容的完整提取。

## 来自章节: 第四章  投标人应当提供的资格、资质性及其他具有类似效力要求的相关证明材料

评分要求提取如下：

1. 资格审查项1  
   - 条件：投标人应具有独立承担民事责任的能力  
   - 证明材料：营业执照（副本）或法人登记证（副本）或其他能够证明投标人具有独立承担民事责任能力的相关证明材料复印件  
   - 备注：营业执照或法人登记证或其他证明材料载明有期限的，应在有效期限内。

2. 资格审查项2（书面声明材料）  
   2.1 具有良好的商业信誉和健全的财务会计制度的证明材料  
   - 提供内容：投标人具有良好的商业信誉和健全的财务会计制度的书面声明材料。  
   - 说明：按招标文件第五章的内容及要求提供书面声明材料。
   
   2.2 具有履行合同所必需的设备和专业技术能力  
   - 提供内容：投标人具有履行合同所必需的设备和专业技术能力的书面声明材料。  
   - 说明：按招标文件第五章的内容及要求提供书面声明材料。
   
   2.3 缴纳税收和社会保障资金的证明材料  
   - 提供内容：依法缴纳税收和社会保障资金良好记录的书面声明材料。  
   - 说明：按招标文件“投标人声明”的内容提供书面声明材料。
   
   2.4 参加采购活动前三年内，在经营活动中没有重大违法记录  
   - 提供内容：书面声明材料证明无重大违法记录。  
   - 额外核查：代理机构在“信用中国”和“中国政府采购网”网站进行核查，并书面反馈评标委员会。如有必要，评标委员会可进行进一步查询。  
   
   2.5 未被列入失信被执行人、重大税收违法失信主体、政府采购严重违法失信行为记录名单  
   - 提供内容：书面声明材料证明未列入上述名单。  
   - 额外核查：代理机构利用指定网站相应查询渠道核查并书面反馈评标委员会。  
   
   2.6 参加采购活动前三年内，投标人及其现任法定代表人、主要负责人没有行贿犯罪记录  
   - 提供内容：书面声明材料证明无行贿犯罪记录。  
   - 备注：投标人可自行根据中国裁判文书网核查，无需提交原始查询结果。虚假声明将按招标文件规定处理。  
   
   2.7 被列入四川农商联合银行采购“黑名单”的企业、自然人及该自然人实际控制的企业不得参与本项目  
   - 提供内容：相关书面声明材料。  
   - 额外核查：代理机构根据采购“黑名单”核查投标人及相关实际控制人和授权代表，并书面反馈评标委员会。

3.2 投标人声明
