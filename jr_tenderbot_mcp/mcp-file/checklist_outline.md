# 商务部分评分

- [ ] 提供并确保以下资格及合规情况，保证资格审查通过，获得评分资格：  
  - 独立承担民事责任能力（有效营业执照或法人登记证书副本）  
  - 具有良好的商业信誉及健全财务会计制度（书面声明）  
  - 具备履行合同所必需的设备和专业技术能力（书面声明）  
  - 依法缴纳税收和社会保障资金，保持良好记录（书面声明）  
  - 近三年（含）内无重大违法记录（书面声明及信用网站核查）  
  - 不列入失信被执行人、重大税收违法失信主体、政府采购严重违法失信名单（书面声明及指定网站核查）  
  - 近三年内无行贿犯罪记录（投标人及主要负责人）  
  - 非四川农商联合银行采购“黑名单”企业、自然人及其控制企业  
  - 非联合体投标（本项目不接受联合体）  
  - 持有效劳务派遣经营许可证及人力资源服务许可证  
  - 遵守不得转包、不允许分包等合同要求  

- [ ] 投标保证金按招标要求足额缴纳，且无以下情形以保证金不予退还：  
  - 投标截止后撤回、修改、替换投标文件  
  - 放弃中标候选资格或中标资格  
  - 不签合同或不交履约保证金  
  - 违法违规、提供虚假材料等严重违规行为  

- [ ] 投标文件需符合格式及提交要求，确保符合性审查：  
  - 投标文件密封、签署、盖章符合招标文件要求  
  - 投标文件实质性响应采购需求，隐含无负偏离，且投标人声明真实有效  
  - 投标文件中语言版本（中文）完整，主要材料有中英文对应或翻译  
  - 提供法定代表人授权书及身份证明材料  
  - 投标有效期不少于90天  
  - 投标报价符合“一报”原则，同一项目仅有一个报价；报价不得超过预算且为实质性报价  
  - 报价明显低于其他合格投标人时，提供合理书面说明及相关证明材料  

- [ ] 投标文件中项目应答表需逐条响应招标文件技术、服务、商务及合同条款要求：  
  - 不得存在实质性负偏离，负偏离视为无效投标  
  - 应答表每页加盖投标人印章  
  - 对优于事项明确填写正偏离及说明，任何虚假应答将取消资格并罚没保证金  

- [ ] 投标人声明承诺符合所有法律法规，不存在以下情形（否则无效）：  
  - 重大违法，包括刑事处罚、责令停产停业等行政处罚  
  - 行贿犯罪记录  
  - 被行政主管部门禁止采购参与  
  - 不具备健全财务制度或不缴纳社保税记录  
  - 多个单位负责人相同或存在控股管理关系的供应商不得投标同一合同  

- [ ] 针对供应商及其人员的纪律要求严格遵守，违法违规行为直接导致投标无效或取消中标资格，包括但不限于：  
  - 提供虚假材料  
  - 串通投标、围标等不正当竞争  
  - 提供非法利益或行贿  
  - 中标后拒绝履约  

- [ ] 保证投标人所属项目团队管理人员及服务人员符合资格，且提供合规证明材料（身份证、证书、缴纳养老保险或劳动合同等复印件及查询截图加盖公章）

# 技术部分评分

- [ ] 履约能力（共43分）  
  - 类似项目业绩（25分）：自2022年1月1日起，每提供一个类似项目业绩得5分，最高25分；须提供合同关键页、增值税发票复印件及佐证材料，均加盖公章。  
  - 质量管理体系认证及相关管理体系证书（最多18分）：分别为质量、环境、职业健康安全、信息安全、食品安全、社会责任认证，每项证书3分，提供证书复印件加盖公章。

- [ ] 履约质量（共27分）  
  - 实施团队资质（12分）：  
    - 项目负责人具有人力资源管理师证书得3分  
    - 其他管理人员中三级及以上企业人力资源管理师每人2分，最多4分  
    - 服务人员中一级劳动关系协调师每人3分，二级每人2分，最多5分  
    - 提供身份证、资格证书复印件、查询截图并加盖公章；项目负责人另附养老保险证明或劳动合同复印件。  
  - 服务方案（15分）：  
    - 人员管理制度（5项）：完全符合得5分，每缺1项扣1分，每项内容错误或不满足扣0.5分，最低为0分  
    - 派遣外包方案（6项）：完全符合得10分，每缺1项扣2.5分，每项错误扣1.25分，最低为0分  

- [ ] 劳务派遣及外包服务要求实质响应（均为必需打分项，影评分数，实质性要求必须满足，偏离为无效）：  
  - 劳务派遣人员岗位管理（驾驶员、设备维护、打字员等岗位）：费用固定，管理费报价不超过30元/月/人  
  - 劳务外包费用管理费要求：大堂经理、网点营销助理等不高于200元/月/人，其他不高于120元/月/人  
  - 3个工作日内提供符合要求的劳务人员  
  - 入职后每季度进行不少于一次职业道德、安全消防、银行服务礼仪等培训  
  - 符合法律法规用工模式，按时发放薪资，缴纳社保，办理劳动合同及变更手续  
  - 制定并提供完备的规章制度及服务方案（外包方案、应急预案等）  
  - 建立完善的服务及项目规范，含人员配备、服务评价、风险控制、投诉处理、业务考核和数据管理  
  - 不得出现因服务质量引起的投诉或媒体曝光  
  - 人力、设备、经营场所固定且满足业务需求  

- [ ] 关键岗位人员资格具体要求（作为评分及服务合规基础）：  
  - 大堂经理学历、年龄、身高、工作经验、证件材料等资格满足招标文件要求  
  - 厨师须持有效厨师证和健康证  
  - 服务人员应持有效健康证  

- [ ] 服务标准及考核管理（含评分点及违约扣分依据）：  
  - 供应商负责派遣外包人员人事管理及工资社保缴纳  
  - 建立人员档案和入职手续管理  
  - 配备专职管理人员协助法规咨询及采购人协调  
  - 劳务人员遵守采购人规章制度并接受考核，考核结果作为奖励处罚依据  
  - 供应商负责劳资纠纷处理及法律诉讼  
  - 接受采购人检查与考核  
  - 定期职业道德及岗位技术培训  
  - 提供符合用人需求的候选人，15日内响应  
  - 不得将同一劳务人员派遣至其他单位  
  - 劳动合同变更解除需与采购人协商  
  - 负责党员党组织关系管理  
  - 实时收集采购人反馈并调整  

- [ ] 供应商须具备完善的运营架构及团队管理：  
  - 至少3人团队，包括1名项目负责人，负责沟通及管理  
  - 建立员工管理、工资及考核制度  
  - 合同签订后一周内团队入场熟悉工作  
  - 确保团队人员在岗稳定，避免频繁无故调整  
  - 配合采购人完成临时交办工作  
  - 经验丰富者岗位可适当调整  
  - 服务方案及团队均必须严格符合采购人需求  

- [ ] 遵守所有强制性及行业标准，产品及服务符合国家认证要求，并能提供相关证明材料

# 价格部分评分

- [ ] 投标报价满足以下要求，保证价格评分满分及报价有效：  
  - 投标总价为全部工作内容报价，实行“一报价”原则，不允许选择或可调整的报价  
  - 报价精度最多为小数点后两位，超出部分四舍五入  
  - 报价不得超过采购预算和最高限价  
  - 报价低于其他有效投标人报价50%及以下时，需书面合理性说明并提供证明，否者视为低于成本报价，投标文件废标  
  - 按最低合格报价进行比价得分：得分=最低合格报价/投标报价×30分，价格分满分30，最低0分  
  - 报价确认后生效，未按规定时间确认报价视为无效投标  
  - 驾驶员、设备维护、打字员等岗位劳务派遣管理费报价不超30元/月/人（不可竞争费用固定报价）  
  - 劳务外包管理费不高于规定上限（大堂经理等岗位不超过200元/月/人，其他岗位不超过120元/月/人）  

- [ ] 履约保证金及扣罚条件（与价格评价关联）：  
  - 履约保证金5万元，合同完成并验收合格后无息退还  
  - 履约保证金不退还条件包括但不限于：  
    - 单方不履行合同（非不可抗力）  
    - 故意降低配置或标准造成损害  
    - 30日内累计3次及以上服务质量问题，按情况扣除最高不超过50%保证金  
  - 履约保证金扣除可依据合同签订的服务考评结果执行
