    - [ ] 提交的投标文件必须为含税总价报价，涵盖完成本项目所需的一切费用。  
    - [ ] 仅允许提供一个固定报价，任何形式的选择性报价或报价调整均视为无效投标。

    - [ ] 报价金额须精确到小数点后两位，超出部分采用四舍五入原则处理。

    - [ ] 投标报价不得高于采购预算人民币1924.44万元及最高限价1924.432万元，超出则投标文件作无效处理。  
  - 报价低于其他有效投标人报价50%及以下时，需书面合理性说明并提供证明，否者视为低于成本报价，投标文件废标  
    - [ ] 若投标报价低于其他有效投标人平均报价50%及以下，投标人须在评标委员会要求时提供书面合理性说明和相关证明材料。  
    - [ ] 未能合理说明报价合理性的，视为低于成本报价，做废标处理。  
  - 按最低合格报价进行比价得分：得分=最低合格报价/投标报价×30分，价格分满分30，最低0分  
    - [ ] 报价得分依据评标基准价与投标报价计算，最高30分，最低0分。

    - [ ] 价格经投标人盖章确认后生效，未在规定时间内完成确认，投标视为无效。
